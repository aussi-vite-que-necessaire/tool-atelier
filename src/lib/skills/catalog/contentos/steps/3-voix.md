# 3. Voix — rédiger dans le ton, sans mise en page

But : transformer le plan validé en **texte rédigé**, dans la **voix** choisie — **sans** encore
toucher à la mise en page.

1. Charge la voix choisie à l'étape 1 (`content` de la voix, via `list_voices`).
2. Rédige le post **en suivant le plan** : une section du plan → un passage rédigé. Garde la
   **structure** du plan.
3. Travaille les **formules** : tournures, rythme, vocabulaire, niveau de langue — pour que le
   texte sonne vraiment dans la voix. C'est ici qu'on soigne le ton, pas le fond (déjà figé).
4. **Pas de cosmétique** à ce stade : pas d'emojis, de hashtags, de puces décoratives ni de
   sauts de ligne « LinkedIn ». Juste un texte juste et bien écrit.

Itère avec l'humain si la voix n'est pas encore satisfaisante.
