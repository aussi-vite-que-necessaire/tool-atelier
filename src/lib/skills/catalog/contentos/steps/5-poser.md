# 5. Poser le post

But : déposer le post final dans l'outil pour que l'humain le voie et le pilote.

1. Crée le post : `create_post` avec un `title` court et descriptif et le `content` final
   (le texte mis en page à l'étape 4).
2. **Montre le post final à l'humain** dans la conversation (le texte tel qu'il sera publié).
3. Confirme qu'il est posé dans Contentos : l'humain peut le retrouver, le relire, le planifier
   ou le publier depuis l'outil.

Ne publie pas toi-même : la publication (immédiate ou planifiée) reste le choix de l'humain.
La partie **visuel** (image/template d'accompagnement à partir du `visualIntent` du format)
est une itération ultérieure, hors de ce squelette.
